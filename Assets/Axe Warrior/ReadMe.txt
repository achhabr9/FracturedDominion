FBX File Structure
The FBX file is named "Axe_Warrior_Full_Set" and includes the character rig along with all three armor sets.

Armor Set Naming Conventions
The armor sets are prefixed for easy identification:
• A1_ → Armor Set 1
• A2_ → Armor Set 2
• A3_ → Armor Set 3
